Projects

Particle Prodigy


A subatomic particle based simulation game created for Jam for a Cause. The game is a heavily abstracted simulation of how protons, neutrons, and electrons, as well as variously charged atoms interact. It is more of a proof of concept that can easily be expanded upon into a fully fleshed out puzzle game. I programmed the physics and interactions between subatomic and atomic particles in C#.

Gravity Golf


A golf game, but set in space, where you must use gravitational fields of planets to predict the ball's trajectory. Created as an academic project on a team of 4, I worked on the physics system which simulates a golf ball interacting with the gravitational pull of planets. I also Created an external level editor.
